Loyalty App Backend

 Live Demo

http://loyaltybackend.gt.tc

 Overview

This is a backend API for a loyalty points system. It allows users to register, log in, and manage reward points.

 Tech Stack

- PHP
- MySQL

 Features

- User registration & login
- Add points
- Get user points
- REST-style API endpoints

🗄 Database

Import "db.sql" into MySQL.

 How to Run

1. Place files in a PHP server (XAMPP / InfinityFree)
2. Import database
3. Update "db.php" credentials

🔒 Note

Sensitive credentials are removed for security.
